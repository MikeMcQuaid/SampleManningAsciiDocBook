## Usage
Clone this repository (or a fork) into the `./input/` subdirectory of a local [ManningAsciiDoc](https://github.com/mikemcquaid/ManningAsciiDoc) clone and use it as a base for your book.

You can read all the sample AsciiDoc source files inline on GitHub.
